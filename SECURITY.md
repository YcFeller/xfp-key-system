# 安全漏洞报告

**项目：** XFP Activation Key System（XFP 密钥获取系统）  
**维护者主页：** https://github.com/YcFeller  

若你发现与本项目相关的安全漏洞，请通过私下渠道联系维护者（例如 GitHub 安全公告、私密 Issue，或你在 [YcFeller](https://github.com/YcFeller) 主页公开的联系方式），**请勿**在公开 Issue 中直接披露可利用细节。

请尽量包含：

- 受影响版本或提交
- 复现步骤与影响范围
- 如有建议修复思路可一并说明

维护者会在合理时间内评估与回复。感谢负责任的披露。
